# Agency Platform Component

A comprehensive agency website and admin dashboard with a modern SaaS aesthetic.

## Features

- **Public Website**: Home, Services, Portfolio, Contact, Login
- **Admin Dashboard**: Stats, Messages, Projects Management
- **Design System**: Dark/Light mode, Glassmorphism, Framer Motion animations
- **Tech Stack**: React, Tailwind CSS, Recharts, Lucide Icons

## Usage

```tsx
import { AgencyPlatform } from '@/sd-components/b2420f91-dc8f-4f0f-8a19-ea86550615fa';

function App() {
  return <AgencyPlatform />;
}
```

## Props

This component functions as a full-page application and does not currently accept external props for configuration. Configuration is handled internally via constants and context.
